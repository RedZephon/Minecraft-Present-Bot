# MC Presence Wiki

Welcome to the MC Presence documentation. MC Presence is a self-hosted Minecraft presence bot with a real-time web dashboard and AI-powered chat.

## Pages

- **[Getting Started](Getting-Started)** — First-time setup walkthrough
- **[Self-Hosting (Node.js)](Self-Hosting-Node.js)** — Run directly on your computer or server
- **[Docker Deployment](Docker-Deployment)** — Run in a Docker container (recommended for Unraid, Synology, etc.)
- **[VPS Deployment](VPS-Deployment)** — Deploy to a cloud VPS (Ubuntu/Debian)
- **[Configuration](Configuration)** — All settings explained
- **[AI Chat Modes](AI-Chat-Modes)** — How the AI assistant works in each mode
- **[CobbleBridge](CobbleBridge)** — Optional Paper plugin for virtual bot support

## Requirements

- **Node.js 18+** (if running directly)
- **Docker** (if using containers)
- A Minecraft server to connect to
- A Microsoft account with Minecraft: Java Edition (per session)
- An [Anthropic API key](https://console.anthropic.com/) (only if using AI chat features)
